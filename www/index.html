Conway's Game of Life

Click on the space to toggle cells between live and dead state. Use the controls below to start or stop the simulation.


This is an in-browser implementation of John Conway's Game of Life. For more information about the Game of Life and for patterns that you can use with this simulator, check out the Life Wiki. You can find the source for this implementation of Life on Github.
